## 二、已修复问题

### 1. 生产构建失败（TypeScript）

- **文件**：`src/components/home/home-play-more.tsx`
- **现象**：`Property 'hot' does not exist on type ...`，`npm run build` 失败
- **修复**：为 `MORE` 数组显式声明 `hot?: boolean` 类型，避免 `as const` 联合类型无法访问可选字段
- **状态**：✅ 已修复，构建 56 路由全部通过

### 2. 自测脚本误连旧 dev 实例

- **现象**：多端口同时 `npm run dev` 时，`/api/me` 偶发 500，内容库同步用例误报失败
- **修复**：`test-supabase-ingest.mjs` / `test-full-app.mjs` 优先检测返回 **401** 的健康实例（端口序：3010 → 3006 → …）
- **建议**：自测前只保留一个服务进程，或使用 `npm run start -p 3010`

### 3. 生产模式登录自测拿不到 devCode

- **现象**：`NODE_ENV=production` 时 `send-code` 不返回 `devCode`，冒烟登录失败
- **修复**：`test-full-app` 在 dev 无 `devCode` 时从 `sms_logs` 表读取验证码（与入库自测一致）

## 七、产品行为说明（非 Bug）

1. **免费版发布包**：日额度 3 次，发布包单次扣 5，新用户无法在一次额度内完成发布包生成；需 Pro 或赠送额度。UI 应引导升级或提示额度不足。
2. **邀请盲盒**：需库中存在带盲盒次数的老用户，自测环境常 SKIP。
3. **多 dev 进程**：热重载期间可能出现短暂 404，自测请用单实例或生产 `start`。
