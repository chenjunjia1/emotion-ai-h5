## 方式一：GitHub 网页上传（本机无 Git 时用）

1. 解压 **`emotion-ai-h5-上传GitHub.zip`**
2. 打开 https://github.com/chenjunjia1/emotion-ai-h5
3. 进入 **`src`** → **Add file → Upload files**
4. 拖入解压后的 **`src` 整个文件夹**（覆盖更新）
5. 若改过根配置，再在仓库**根目录**上传：`package.json`、`package-lock.json`、`next.config.ts`
6. Commit 说明（可复制）：

```
feat: 首页Banner文案、会员页UI、盲盒额度提示修复

- 顶栏 ai短视频运营，轮播第1张新文案
- Pro 轮播第3张，会员订阅页分层皇冠
- 盲盒次数与总额度分开提示
- 邀请进度 key 修复，测试验证码 1234
```

7. 打开 https://vercel.com → 项目 **emotion-ai-h5** → **Deployments** → 等 **Ready**（约 2～5 分钟）
8. 无痕窗口打开线上地址，**Ctrl+F5** 验收

## 方式二：本机有 Git 时（一条命令推送）

在项目目录 `E:\emotion-ai-h5` 打开终端：

```bash
git add -A
git commit -m "feat: 首页Banner与会员UI、盲盒额度修复"
git push origin main
```

Vercel 会自动部署。

## Vercel 环境变量（确认一次）

**Settings → Environment Variables** 建议保留：

- `NEXT_PUBLIC_BACKEND_MODE=server`
- `SMS_PROVIDER=dev`（正式短信改 `aliyun`）
- Supabase、`SESSION_SECRET`、DeepSeek 等（见 `.env.local`，勿上传密钥到 GitHub）

改完变量后点 **Redeploy**。
