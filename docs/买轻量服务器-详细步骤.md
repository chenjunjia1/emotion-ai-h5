# 阿里云轻量应用服务器 · 购买与备案关联（详细步骤）

用于部署「情绪价值助手」并在 ICP 备案里选择 **云产品实例**。

---

## 第一部分：购买轻量服务器

### 1. 打开购买页

任选一种：

- 阿里云控制台顶部搜索：**轻量应用服务器**
- 或打开：https://www.aliyun.com/product/swas
- 控制台左侧：**轻量应用服务器** → **服务器** → **创建服务器**

### 2. 地域（和备案一致）

| 备案填的地区 | 服务器建议选 |
|--------------|--------------|
| 浙江 / 杭州 | **华东1（杭州）** |
| 上海 | **华东2（上海）** |
| 广东 / 深圳 | **华南1（深圳）** |
| 北京 | **华北2（北京）** |

**原则：** 备案主体在哪个省，优先选 **同区域或邻近区域** 的机房。

### 3. 镜像（系统）

- 选 **系统镜像**
- 选 **Ubuntu**
- 版本：**Ubuntu 22.04 64位**

（不要选「应用镜像」里的 WordPress、宝塔等，除非你很熟悉。）

### 4. 套餐

- **2核 CPU / 2GB 内存 / 40GB 系统盘**（最便宜档通常够用）
- 带宽：**3M 或 5M** 即可（MVP）
- 时长：先 **1 个月** 或 **1 年**（按预算）

### 5. 购买方式 / 名称

- 实例名称：可填 `emotion-ai`（随意）
- 设置 **root 密码**（记下来！SSH 登录要用）
- 或选 **密钥对**（新手用密码更简单）

### 6. 确认订单并支付

支付完成后，等 **1～3 分钟** 状态变为 **运行中**。

### 7. 记下两个信息（重要）

在 **轻量应用服务器 → 服务器列表** 里看到：

| 信息 | 示例 | 用途 |
|------|------|------|
| **公网 IP** | `47.98.xxx.xxx` | 备案、DNS、SSH |
| **实例 ID / 名称** | 一串 ID | 备案下拉里选 |

---

## 第二部分：放行端口（安全 / 防火墙）

在轻量服务器控制台 → 点你的实例 → **防火墙** / **安全**:

确保有规则（没有就添加）：

| 端口 | 说明 |
|------|------|
| **22** | SSH 远程连接 |
| **80** | 网站 HTTP |
| **443** | 网站 HTTPS |
| **3000** | 临时测试 Next.js（可选，正式上线用 Nginx 可不开外网 3000） |

---

## 第三部分：回到 ICP 备案选实例

### 1. 打开备案页

https://beian.aliyun.com → 你的备案订单 → **继续填写**

### 2. 网站接入信息

| 项目 | 选择 |
|------|------|
| **云服务** | **轻量应用服务器** |
| **云产品实例** | 点下拉框 / 搜索 |

### 3. 正确选项长什么样

应看到类似：

- `轻量应用服务器 / 47.98.xxx.xxx`
- 或带地域 + IP 的实例名

**选带「轻量」和「你的公网 IP」的那条。**

### 4. 不要选什么

❌ **智能建站 / WS202605… / 116.62.172.xxx**  
→ 那是「万小智 AI 建站」，不是轻量 ECS，**不要选**。

### 5. 若下拉仍是「暂无数据」

1. 确认轻量服务器状态是 **运行中**
2. 确认和备案是 **同一个阿里云账号** 登录
3. 刷新备案页面 / 重新登录
4. 等 5～10 分钟再试（新购实例同步有延迟）
5. 备案页点 **「使用已有实例」** 或系统推荐里 **切换产品类型为轻量**

### 6. 保存

- **保存草稿**
- **完成填写** → 继续后续核验

## 第四部分：买好后部署网站（备案可并行）

SSH 连接（控制台点 **远程连接** 即可）：

```bash
# 安装 Node
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs git
sudo npm install -g pm2

# 拉代码（改成你的仓库地址）
cd /var/www
sudo git clone https://github.com/chenjunjia1/emotion-ai-h5.git
cd emotion-ai-h5

# 环境变量
sudo nano .env.local
# 粘贴 DEEPSEEK + Supabase 四个变量

npm install
npm run build
pm2 start npm --name emotion-ai -- start
pm2 save
```

浏览器测试：`http://你的公网IP:3000`

## 常见问题

**Q：轻量和 ECS 有什么区别？**  
A：对 MVP 来说，轻量更简单、便宜，够用了。

**Q：备案必须买服务器吗？**  
A：大陆备案通常要关联 **大陆境内的云服务器实例**，轻量算一种。

**Q：买在香港地域可以吗？**  
A：备案大陆网站 **不建议**，选 **大陆地域**（华东/华北/华南等）。

**Q：已有「智能建站」还要买轻量吗？**  
A：**要。** 智能建站跑不了咱们的 Next.js API，需单独买轻量。
