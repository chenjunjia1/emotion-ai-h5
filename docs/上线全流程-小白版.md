# 情绪价值助手 · 上线全流程（小白版）

你已经完成：**买了域名 emovalue.top**  
还差两件事：**① 把网站放到公网  ② 让域名指向这个网站**

最终效果：手机打开 **https://emovalue.top** 就能用你的 H5。

---

# 第一部分：把网站部署到 Vercel（约 30 分钟）

## 1.1 注册 GitHub（存代码的网盘）

1. 打开 [https://github.com](https://github.com)
2. 点 **Sign up** 注册（可用邮箱）
3. 登录成功即可

## 1.2 在 GitHub 新建仓库

1. 登录后点右上角 **+** → **New repository**
2. **Repository name** 填：`emotion-ai-h5`
3. 选 **Private**（私有，别人看不到代码）
4. **不要**勾选 README
5. 点 **Create repository**

## 1.3 把电脑上的项目上传到 GitHub

**不要上传这些文件夹/文件**（很重要）：

- `node_modules` 文件夹
- `.next` 文件夹
- `.env.local` 文件（里面有密钥！）
- `.npm-cache` 文件夹

**只上传其余所有文件**（含 `src`、`package.json` 等）。

### 上传方法（不用装 Git，用网页）：

1. 在电脑上打开 `e:\emotion-ai-h5`
2. 选中除了上面排除项以外的**所有文件和文件夹**
3. 右键 → 发送到 → 压缩(zipped)文件夹，得到 `emotion-ai-h5.zip`
4. 回到 GitHub 刚建的仓库页面
5. 点 **uploading an existing file** 或 **Add file → Upload files**
6. 把 zip **解压后的内容**拖进去（或拖单个文件，不要拖 zip 本身）
7. 页面底部 **Commit changes**

> 若 zip 太大：用 [GitHub Desktop](https://desktop.github.com/) 更简单，安装后 File → Add Local Repository 选 `e:\emotion-ai-h5`，再 Publish。

## 1.4 注册并登录 Vercel

1. 打开 [https://vercel.com](https://vercel.com)
2. 点 **Sign Up** → 选 **Continue with GitHub**（用 GitHub 登录）
3. 授权 Vercel 访问 GitHub

## 1.5 在 Vercel 导入项目

1. 登录 Vercel 后点 **Add New… → Project**
2. 列表里找到 **emotion-ai-h5** → 点 **Import**
3. **Framework** 应自动是 **Next.js**，不用改
4. 先**不要**点 Deploy，往下找到 **Environment Variables**

## 1.6 填写 4 个环境变量（和本地一样）

打开你电脑上的 `e:\emotion-ai-h5\.env.local`，把下面 4 项**逐个**抄到 Vercel：

| 名称（Name） | 值（Value）从 .env.local 复制 |
|--------------|-------------------------------|
| `DEEPSEEK_API_KEY` | sk-开头那一串 |
| `NEXT_PUBLIC_SUPABASE_URL` | https://xxx.supabase.co |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | eyJ 开头 |
| `SUPABASE_SERVICE_ROLE_KEY` | eyJ 开头（service_role） |

每一项点 **Add**，三个环境 **Production / Preview / Development** 都勾选。

## 1.7 点击 Deploy

1. 点 **Deploy**
2. 等 1～3 分钟，出现 **Congratulations**
3. 点 **Visit**，会打开类似：`https://emotion-ai-h5-xxxxx.vercel.app`
4. 用手机或电脑试一下：能打开首页、能生成 → **第一部分完成**

# 第二部分：绑定域名 emovalue.top（约 15 分钟 + 等待）

## 2.1 在 Vercel 添加域名

1. 在 Vercel 打开你的项目 **emotion-ai-h5**
2. 顶部点 **Settings**
3. 左侧点 **Domains**
4. 输入框填：`emovalue.top` → 点 **Add**
5. （可选）再添加 `www.emovalue.top`

Vercel 会显示 **Invalid Configuration** 和几行 **DNS 记录**，先别关页面。

常见记录（**以你页面上显示的为准**）：

| 类型 | 主机记录 | 记录值 |
|------|----------|--------|
| A | `@` | `76.76.21.21` |
| CNAME | `www` | `cname.vercel-dns.com` |

把这两行**抄在记事本**里。

## 2.2 在阿里云添加 DNS 解析

1. 打开 [https://dc.console.aliyun.com](https://dc.console.aliyun.com) 或控制台搜 **域名**
2. 左侧 **域名列表** → 点 **emovalue.top**
3. 点 **DNS 修改** 或 **解析设置** 或 **云解析 DNS**
4. 点 **添加记录**，按 Vercel 要求添加：

**记录 1（根域名）：**

- 记录类型：**A**
- 主机记录：**@**（有的界面写「空」或根域名）
- 记录值：**76.76.21.21**（用 Vercel 页面上写的 IP）
- TTL：默认 10 分钟

**记录 2（www，可选）：**

- 记录类型：**CNAME**
- 主机记录：**www**
- 记录值：**cname.vercel-dns.com**

5. 保存

## 2.3 等待生效

1. 回到 Vercel **Domains** 页面，点 **Refresh**
2. 等 10 分钟～2 小时，状态变 **Valid Configuration**（绿勾）
3. 手机浏览器打开：**https://emovalue.top**

# 常见问题

**Q：必须用 GitHub 吗？**  
A：最简单是 GitHub + Vercel。也可用 GitHub Desktop 代替命令行。

**Q：万小智 AI 建站要用吗？**  
A：**不要。** 和本项目无关。

**Q：.env.local 要上传吗？**  
A：**绝对不要。** 只在 Vercel 网页里填环境变量。

**Q：DNS 一直 Invalid**  
A：检查主机记录 `@` 和 IP 是否填对；多等一会再 Refresh。
