# 深圳个体工商户 — 网上办理步骤（照着点）

> 我无法替你登录政务网或提交申请（要用你的身份证、人脸识别、支付宝实名）。  
> 下面按 **2025～2026 常用渠道** 写，你按顺序做即可。

---

## 一、办之前准备（今天就能准备）

| 材料 | 说明 |
|------|------|
| **身份证** | 经营者本人，在有效期内 |
| **手机号** | 能收验证码 |
| **深圳经营场所** | 网店可用「网络经营场所」或挂靠地址（需符合当地政策；无地址可先问代办或政务热线） |
| **经营范围** | 建议含：`软件开发`、`信息技术咨询服务`、`广告设计、制作、代理`、`市场营销策划`、`互联网销售（除销售需要许可的商品）` 等（以系统可选为准） |
| **名称** | 想好几个，如：`深圳市南山区XX信息咨询服务部（个体工商户）` |

**费用参考：** 执照本身工商登记多 **0 元**；若用地址挂靠/代办另算。

## 二、推荐办理入口（电脑）

### 入口 A：广东政务服务网（最常用）

1. 浏览器打开：**https://www.gdzwfw.gov.cn**
2. 右上角 **登录**（用你个人账号，没有就 **注册**）
3. 登录后右上角地区选 **深圳市**
4. 顶部搜索：**个体工商户设立登记**
5. 选 **深圳市** 的实施清单 → **在线办理**

或直接打开办事指南（可能随政策更新）：

**https://www.gdzwfw.gov.cn/portal/simple-guide/11440300MB2C927392344012501600101**

### 入口 B：深圳市场监管「开办企业一窗通」

1. 打开：**https://amr.sz.gov.cn/psout/**  
   或官网：**http://amr.sz.gov.cn** → 找 **开办企业 / 商事登记**
2. 用 **广东政务服务网账号** 登录（同一套账号）
3. 点 **「我要申请」** → **设立** → **个体工商户**

### 入口 C：手机微信（不用电脑时）

微信搜公众号：**深圳市场监管** 或 **商事登记微申报** → 按提示办设立。

## 三、网上填写（设立登记）— 逐步点

### 第 1 步：名称自主申报

1. 进入设立流程后，先做 **名称申报**
2. 行政区划：**深圳市** + 区（如南山区/宝安区，按你实际）
3. 字号：如 `XX信息` / `XX科技`（2～4 字常见）
4. 行业：如 `咨询服务` / `信息技术`
5. 组织形式：**个体工商户**
6. 提交名称，等 **名称保留** 成功（名称核准）

### 第 2 步：填写设立信息

| 字段 | 建议填法 |
|------|----------|
| 经营者 | 你本人姓名、身份证号 |
| 出资额 | 个体户常填 **1 万元** 或按系统默认 |
| 经营范围 | 勾选上面建议项 + 系统里与「软件/互联网/咨询」相关的 |
| 经营场所 | 有实体店填地址；纯线上可查是否支持 **网络经营场所** 表述 |
| 联络员 | 一般填经营者本人 |

### 第 3 步：实名认证（人脸识别）

- 经营者本人按页面提示做 **人脸识别**
- 账户可信等级需 **L2** 及以上（在广东政务服务网完成实名认证）

### 第 4 步：电子签名 & 提交

- 下载 **未签名 PDF 申请表**
- 用 **个人银行 U 盾** 或 **CA 数字证书** 电子签名（部分流程支持 **支付宝/微信** 扫码签，以页面为准）
- 所有人签完 → **提交**
- 显示 **提交成功** 后等审核（常见 **1 个工作日**）

### 第 5 步：领照

- **电子营业执照**：支付宝小程序 **「电子营业执照」** 领取
- 或按短信到 **自助机** 打印纸质执照

## 四、执照下来后（对接你的 H5 + 支付宝）

```
① 下载/领取电子营业执照
② 登录 https://b.alipay.com → 用执照对应主体开通商户
③ 商家平台 → 申请开通「手机网站支付」（你已在填的页面）
④ 开放平台应用 → 提交审核上线
⑤ Vercel：PAY_PROVIDER=alipay → Redeploy
```

密钥你本地已配好，**有执照后主要是开通商户 + 产品审核**。

## 五、常见问题

| 问题 | 处理 |
|------|------|
| 没有深圳地址 | 问 **12345** 或政务大厅是否可用网络经营场所；或正规挂靠（慎选诈骗代办） |
| 没有 U 盾 | 看流程是否支持 **移动端签名**；或到 **市场监管局窗口** 现场办 |
| 名称重复 | 换字号再申报 |
| 经营范围不会选 | 以系统勾选为准，优先「软件/信息技术/咨询/互联网销售」 |
| 办完能否开支付宝 | 可以，用 **个体户执照 + 经营者支付宝** 去商家平台 |
