在 Supabase Dashboard → **SQL Editor** → New query，粘贴并执行：

`supabase/migrations/005_v1_product_invite_growth.sql`

**逐步图文说明（005→006→Vercel→自测）**：见 [Supabase迁移005-006-详细步骤.md](./Supabase迁移005-006-详细步骤.md)

## 本迁移会创建/补充

- 用户成长字段：`growth_xp`、`streak_days`、`invite_blind_box_count` 等
- 表 `invite_records`（邀请记录）
- 表 `daily_hot_topics`（每日热点）
- 表 `user_daily_usage`（盲盒/扭蛋每日次数）

## 如何确认成功

执行后无报错，且 Table Editor 能看到上述三张表。

## 未执行时的表现

- 邀请记录列表可能为空或接口报错
- 热点可能仅走本地缓存

执行完成后在 Vercel **Redeploy** 一次。
