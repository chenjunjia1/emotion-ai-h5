# 微信内分享 / 打开 H5 配置说明

## 已做（代码内）

- `src/app/layout.tsx` 已配置 `openGraph`、`metadataBase`（读取 `NEXT_PUBLIC_APP_URL`）。
- 分享标题/描述可在 `layout.tsx` 的 `metadata` 中修改。

## 你需要在公众平台配置

1. **微信公众平台**（服务号/订阅号）→ 设置 → 公众号设置 → 功能设置 → **JS 接口安全域名**、**网页授权域名**（若要做 OAuth 登录再配）。
2. **业务域名**：填写你的 H5 域名（如 `www.emovalue.top`），按提示下载校验文件放到 `public/`。
3. 分享卡片图：在 `public/` 放置 `og-share.png`（建议 1200×630），并在 `layout.tsx` 的 `openGraph.images` 中引用。
