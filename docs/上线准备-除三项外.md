## 你上线前仍需在 Vercel 确认（约 10 分钟）

复制 `.env.local.example` 为本地 `.env.local` 填好后，对照 Vercel **Settings → Environment Variables**：

```env
NEXT_PUBLIC_BACKEND_MODE=server
NEXT_PUBLIC_APP_URL=https://你的正式域名
DEEPSEEK_API_KEY=sk-...
CRON_SECRET=随机32位以上
ADMIN_MOBILES=你的手机号
```

改完 → **Deployments → Redeploy**。

验证：

```bash
npm run check:env
npm run test:full
```
