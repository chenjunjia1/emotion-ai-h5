# 首页 Banner 动图资源（奶油盲盒风轮播）

把动图放到本目录，刷新页面即可自动替换右侧 CSS 趣味插画。

## 推荐文件名

| 轮播页 | 文件名 | 说明 |
|--------|--------|------|
| ① 多平台发片 | `creator-motion.gif` | 手机拍片 / 创作者 / 三平台感 |
| ② 邀请好友 | `invite-motion.gif` | 好友、分享、裂变感，适合 90/00 后 |
| ③ 灵感盲盒 | `blindbox-motion.gif` | 礼盒 / 盲盒开箱类，温柔可爱 |

也支持 **`.webp`**（体积更小）：改 `src/components/home/home-cream-carousel.tsx` 里 `MOTION` 常量后缀即可。

## 动图建议

- 尺寸：约 **240×240** 或 **1:1**，透明底更佳
- 风格：奶油色、珊瑚粉、蜜桃橙，不要荧光刺眼
- 体量：单张建议 **< 800KB**，移动端更流畅
- 气质：像小红书运营位 / 盲盒 App，不要商务 PPT 风

## 未放置动图时

会自动使用 CSS 动效占位（礼盒浮动、好友火箭、扭蛋旋转等），不影响使用。
